* 时间复杂度

	```
	O(1) < O(logn) < (n) < O(nlogn) < O(n^2) < O(n^3) < O(2^n) < O(n!) < O(n^n)
	```

	* 用常数1取代运行时间中的所有加法常数。
		
		```
		const a =1;
		for(let i =0;i<n;i++){ // O(n)
			a+=i; //这就是一个O(1)
		}
		```
	* 在修改后的运行次数函数中，只保留最高阶项。
		
		```
		T = O(n^2+3n+1)
		==
		T = O(n^2)
		```
	* 如果最高阶项存在且不是1，则去除与这个项相乘的常数。

		```
		T = O(3*n^2) == O(n^2)
		
		T = O(2n) != O(n) 错误
		T = O(2n) == O(2n)
		```
	* 得到的最后结果就是大O阶。
